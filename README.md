# Galactic Blog System 🌌

这是一个基于 Markdown 和 星空网状图 的炫酷博客系统。

## 如何使用

### 1. 写博客
直接在 `posts` 文件夹下创建 `.md` 文件即可。

**文件格式示例：**
```markdown
---
title: 我的新文章
date: 2026-01-07
tags: [diary, coding]
---

这里写正文...

支持引用其他文章：
- 使用 [[文件名]] 格式，例如 [[hello-world]]
- 或者标准的 Markdown链接 [显示的文字](hello-world.md)
```

### 2. 更新系统
每次添加或修改文章后，运行根目录下的 `Big_Bang_Update.bat` 脚本。
它会：
1. 扫描所有文章。
2. 生成星空图谱数据。
3. 启动一个本地服务器预览 (http://localhost:8000)。

### 3. 特性
- **星空网状图**：所有文章都是一颗星星，引用关系构成星座。
- **直升机小猫**：当你浏览时，一只直升机小猫会一直陪伴你。
- **沉浸式阅读**：点击星星即可展开阅读。
